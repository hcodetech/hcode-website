Riff : An Audio based Collaboration App

We built an app for teams to collaborate over audio-conferencing with features such as file sharing and whiteboard drawings for startup Riff. For this we did a third party integration with agora's audio platform.

The product included:
  • React native based app,
  • django based backend,
  • frontend animation using lottie,
  • third party Agora's voice conference platform integration

  • Django
  • React
  • Voice Chat
  • Conferencing
