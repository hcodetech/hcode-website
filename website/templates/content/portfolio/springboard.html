SpringBoard.com : First Prototype

We created the first prototype of the complete application for SpringBoard, and the firm raised USD 10M+ in funding. SpringBoard is a review and aggregation site for online courses and MOOCs like coursera, udemy and many other providers.

The prototype included:
  • the backend API in Django,
  • the frontend design in HTML5 (responsive with grid layout system),
  • integrated social APIs like facebook,
  • review and rating system and
  • deployed the API on a cloud server.

We also scrapped the data for the application from multiple sites online (using Python - BeautifulSoup)
