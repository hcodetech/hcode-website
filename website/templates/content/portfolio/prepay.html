Prepay : Pre-payments Platform 

We created the pre-payments web-app platform for startup Prepay. This platform intends to provide a service for customers to prepay securely to their vendors for services later.

The platfrom included:
  • Written in React JS,
  • Supported by Django on backend,
  • Responsive front-end design in HTML5,
  • Payments Integration : Stripe

  • Django
  • ReactJS
  • HTML5
  • Stripe
